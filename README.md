# 🎓 EduScope: Student Performance & Placement Analytics

## 📌 Overview

EduScope is a beginner-friendly data analytics project that explores student academic performance and placement trends using Python.

The project focuses on analyzing how factors such as attendance, study hours, internships, projects, and certifications relate to academic performance and placement outcomes through Exploratory Data Analysis (EDA).

---

## 🎯 Objectives

- Analyze student academic performance
- Understand placement trends
- Explore the relationship between attendance and CGPA
- Study salary distribution among placed students
- Generate meaningful business insights using data

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

---

## 📂 Dataset

A synthetic dataset containing **5,000 engineering students** was generated using NumPy and Pandas.

The dataset includes:

- Student Details
- Academic Performance
- Attendance
- Study Hours
- Projects
- Certifications
- Internship Status
- Placement Status
- Company Tier
- Salary

---

## 📊 Analyses Performed

- Student Distribution by Branch
- Gender Distribution
- CGPA Distribution
- Attendance vs CGPA
- Placement by Branch
- Salary Distribution
- Internship vs Placement
- Correlation Heatmap
- Projects vs Salary
- Average CGPA by Branch
- Certifications vs Placement
- Company Tier Distribution

---

## 💡 Key Insights

- Higher attendance is associated with better CGPA.
- Internship experience improves placement outcomes.
- Students with more projects generally receive higher salary packages.
- Certifications positively influence placement opportunities.
- Most salary offers fall between 4–15 LPA.
